# Kayman Srinivasan — Portfolio

AI & Data Science portfolio. Static site, no build step required.

## Structure

```
portfolio/
├── index.html      — markup
├── style.css       — all styling
├── main.js         — neural canvas, cursor, animations
└── assets/
    └── photo.jpg   — your photo (replace this)
```

## Deploy to GitHub Pages

1. Create a repo named exactly: `kaymansrinivasan.github.io`
2. Upload all files (keep folder structure)
3. Go to Settings → Pages → Source: `main` branch, root `/`
4. Save — live at `https://kaymansrinivasan.github.io`

## Before deploying — update these

- `index.html` line with `kayman@email.com` → your real email
- `assets/photo.jpg` → your actual photo file
- Add GitHub repo links to each project's `href`
- Update internship company name in the About section once confirmed
